Display copyright information with a shortcode and customize settings in the WordPress dashboard.

== Description ==

This plugin allows you to easily display copyright information on your WordPress site using a shortcode. You can customize the copyright year, site title, and "Powered by" text through the WordPress dashboard.

Features:

- Simple shortcode to display copyright information.
- Customize the copyright year and site title.
- Editable "Powered by" text with a customizable link.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'Copyright Settings' under the 'Settings' menu in the WordPress dashboard to configure the plugin settings.
4. Use the `[vs_copyright]` shortcode to display the copyright information on your site.

== Frequently Asked Questions ==

= How do I customize the copyright information? =

You can customize the copyright year, site title, and "Powered by" text through the 'Copyright Settings' page in the WordPress dashboard.

= How do I display the copyright information on my site? =

Use the `[vs_copyright]` shortcode in your posts, pages, or widgets to display the copyright information.
